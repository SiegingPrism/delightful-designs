# Core Features

Each feature is mapped to: **(Page) — What it does — Why it earns its place**.

## Tier 0 — The Core Loop (must-have, shipped)
| Feature | Page | Why |
|---|---|---|
| Capture a task in <5s | Tasks, Dashboard | Friction kills the loop |
| Complete → earn XP → see level bar move | Everywhere | Dopamine hit per action |
| Daily habit check-in with streak | Habits, Dashboard | Compounding behavior |
| Pomodoro focus timer with auto-XP | Focus | Deep work has to pay |
| Daily Productivity Score | Dashboard | One number that summarizes today |
| AI Coach: 1 nudge / 3 tasks per day | Coach | Removes "what now?" paralysis |

## Tier 1 — Retention drivers
| Feature | Page | Why |
|---|---|---|
| Habit heatmap (30-day) | Habits | Visible progress = motivation |
| Mood + water + steps quick log | Dashboard, Health | Body data feeds coach |
| Rewards & unlockable themes | Rewards | Long-term goals |
| Insights: weekly + monthly trends | Insights | "Am I getting better?" |
| Calendar view of completed work | Calendar | Proof of effort |

## Tier 2 — Differentiators (planned)
| Feature | Why we win |
|---|---|
| **Skill trees** (Focus, Health, Learning, Craft) | No competitor has this — XP routes into a chosen branch |
| **AI-generated daily missions** | Notion/Todoist don't generate work for you |
| **Burnout detection + forced rest day** | Anti-burnout is a moral + retention moat |
| **Adaptive difficulty** | App gets harder/easier with you |
| **Dopamine control mode** | Optional cap on notifications + rewards to reduce addiction loops |

## Tier 3 — Future
- Voice capture ("Hey FlowSphere, log a 25m focus on the API refactor")
- Apple Health / Google Fit auto-sync
- Wearable (Watch) companion: streak + next task on wrist
- Public profile / friend streaks (opt-in only — never the default)

## Explicitly out of scope (v1)
- Team features
- Project hierarchies
- Gantt / timeline views
- Email integration
